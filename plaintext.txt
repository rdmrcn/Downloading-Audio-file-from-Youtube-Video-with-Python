YouTube Audio Downloader Script  
Created by: Reha Demircan  
Date: 24.11  

Description:  
This script allows users to download the audio from a YouTube video.  
It is written in Python and uses the 'pytube' library to fetch the audio stream from the provided video URL.  

Usage Instructions:  
1. Run the script in a Python environment.  
2. Enter the YouTube video URL when prompted.  
3. The audio file will be downloaded to the current working directory.  

Troubleshooting:  
- Ensure you have an active internet connection.  
- Install 'pytube' using `pip install pytube` if not already installed.  
- If errors occur, try upgrading pytube with `pip install --upgrade pytube`.
